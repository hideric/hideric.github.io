CPrimer - 字符串和字符串函数

表示字符串和字符串IO

字符串常量属于静态存储类别，函数中使用字符串常量，只会被储存一次，程序生命期内存在，即使函数调用多次。用双引号括起来的内容被视为指向该字符串储存位置的指针，类似于数组名作为指向数组位置的指针。

printf("%s, %p, %c\n", "We", "are", *"space"); // *"space"的首字符是指针所指向的位置
// 输出：
We, 0x100000fac, s

字符串数组初始化

const char m1[40] = "Limit yourself to one line's worth.";
const char m1[40] = {'L', 'i' ... 'h', '.', '\0'}; // 最后没有这个\0就不是字符串，而是字符数组

指定数组大小时，要确保数组的元素个数至少比字符串长度多1，所有未被使用的元素都被自动初始化为\0。

数组与指针

const char * pt1 = "Something is pointing me."; // 指针形式
const char ar1[] = "Something is pointing me."; // 数组形式
  1. 数组形式中，ar1是地址常量，不能更改ar1，不能用自增运算法，只能+1指向数组下一个元素，自增运算符只能用于变量前；
  2. 指针形式中，程序开始执行时为pt1留出一个储存位置，存放字符串的地址，指针可以使用自增运算符；
  3. 初始化数组把静态存储区中的字符串拷贝到数组中，而初始化指针只把字符串地址拷贝给指针。

字符串的地址

#define MSG "I'm special"
int main() {
 		char ar[] = MSG;
    const char * pt = MSG;
    printf("address of \"I'm special\" is %p.\n", "I'm special");
    printf("address of ar is %p.\n", ar);
    printf("address of pt is %p.\n", pt);
    printf("address of MSG is %p.\n", MSG);
    printf("address of \"I'm special\" is %p.\n", "I'm special");
  	return 0;
}
// 输出：
address of "I'm special" is 0x100000f3d.
address of ar is 0x7ffeefbff50c.
address of pt is 0x100000f3d.
address of MSG is 0x100000f3d.
address of "I'm special" is 0x100000f3d.
  1. 两次printf函数的”I’m special”的地址相同，说明编译器只使用了一个存储位置；
  2. 静态数据使用的内存和ar使用的动态内存不一样；

数组字符串和指针字符串的区别

char heart[] = "I love Tille!";
const char * head = "I love Tille!";

表面上的区别：数组名heart是常量，而指针名是变量；

使用上的区别：

  1. 只有指针表示法可以进行递增操作；

  2. 只能指针指向数组，不能数组指向指针；

    head = heart; // head现在指向数组heart，head指向heart不会导致指向的字符串消失
heart = head; // 不允许，赋值运算符左侧必须是变量

指针限定符最好加上const修饰，否则会有问题：

char * p1 = "Klingon";
p1[0] = 'F';
printf("Klingon");
printf(":Beware the %ss!\n", "Klingon");
// 输出：
Flingon:Beware the Flingons!

编译器可以使用内存中的一个副本来表示所有完全相同的字符串字面量，也就是说，编译器可以用相同的地址替换每个“Klingon”实例。然而把非const数组初始化为字符串字面量不会导致这样的问题，因为数组获得的是原始字符串的副本。

总之，如果不修改字符串，不要用指针指向字符串字面量。

字符串数组

const char *mytalents[5] = {"a", "b", "c", "d", "e"};
char yourtalents[5][40] = {"a", "b", "c", "d", "e"};
  1. mytalents和yourtalents都代表5个字符串
  2. mytalents数组是一个内含5个指针的数组，在系统中共占用40个字节。而yourtalents是一个内含5个数组的数组，每个数组内含40个char类型的值，共占用200字节。
  3. mytalents指向字符串字面量位置，字面量存储在静态内存中。而yourtalents中数组则储存字符串字面量的副本，所以每个字符串储存2次，为字符串数组分配内存使用率较低。
  4. yourtalents中的每个元素大小必须相同（没行长度都是40字节），而mytalents可以想象成不规则的数组。

综上所述，如果要用数组表示一系列待显示的字符串，请使用指针数组，因为它比二维字符数组的效率高。指针数组的缺点是指向的字符串字面量不能更改；而yourtalents中的内容可以更改，所以，如果要改变字符串或为字符串输入预留空间，不要使用指向字符串字面量的指针。

指针和字符串

const char * mesg = "Don't be a fool!";
const char * copy;
copy = mesg;
printf("%s\n", copy);
printf("mesg = %s; &mesg = %p; value = %p\n", mesg, &mesg, mesg);
printf("copy = %s; &copy = %p; value = %p\n", copy, &copy, copy);
// 输出：
Don't be a fool!
mesg = Don't be a fool!; &mesg = 0x7ffeefbff4b8; value = 0x100000f5d
copy = Don't be a fool!; &copy = 0x7ffeefbff4b0; value = 0x100000f5d

可以看出copy=mesg只是把mesg的值（指向字符串的地址）赋给copy，并没有拷贝字符串，这两个指针都指向同一个字符串地址。

2.字符串输入

2.1.分配空间
char *name;
scanf("%s", name);

虽然可能通过编译，但是scanf函数将输入的信息拷贝至参数指定的地址上时，这个参数是个未初始化的指针，可能导致name指向任何地方。所以声明时显示的指明数组的大小。

2.2.不幸的gets()函数

gets()函数用于解决scanf(“%s”)只能读取一个单词的问题，gets()函数遇到换行符终止，常常配合puts()函数使用。

gets()函数的问题在于无法检查数组是否装得下输入行。字符串过长，会导致缓冲区溢出，访问未分配内存或擦除其他数据，程序提示“Segmentation fault”表示在访问未分配的内存。

2.3.gets()的替代品

使用fgets()函数，第二个参数限制读入的字符数，第3个参数指明读入的文件（键盘读入使用stdin）。

fputs()函数的第2个参数指明要写入的文件，如果要显示，应使用stdout未参数。

#define STLEN 14
int main(void) {
    char words[STLEN];
    puts("Enter a string, please.");
    fgets(words, STLEN, stdin);
    printf("Your string twice (puts(), then fputs()):\n");
    puts(words);
    fputs(words, stdout);
    puts("Enter another string, please.");
    fgets(words, STLEN, stdin);
    printf("Your string twice (puts(), then fputs()):\n");
    puts(words);
    fputs(words, stdout);
    puts("Done.");
    return 0;
}
// 输出：
Enter a string, please.
apple pie
Your string twice (puts(), then fputs()):
apple pie

apple pie
Enter another string, please.
strawberry shortcake
Your string twice (puts(), then fputs()):
strawberry sh
strawberry shDone.

  1. puts()函数会在没有填满缓冲区的字符串后面添加换行符，而fputs()函数不会。

  2. fputs()函数返回指向char的指针。读到文件结尾会返回特殊的指针：空指针（null pointer），C语言中用宏NULL来代替。

    // 循环读取不会受限制于字符串长度，读取文件结尾校验
while(fgets(words, STLEN, stdin) != NULL && words[0] != '\n'){
		fputs(words, stdout);
}

  3. 空字符时整数类型，空指针式指针类型。都可以用数值0表示。空字符是一个字符，占1字节；空指针是一个地址，通常占4字节。

gets_s()函数

gets_s()函数只从标准输入中读取数据，所以不需要底3个参数。

如果gets_s()函数遇到换行符，会丢弃它而不是储存它。

只要输入行未超过最大字符数，gets_s()和gets()几乎一样，超过最大字符数，gets_s()先把目标数组中的首字符设置为空字符，读取并丢弃随后的输入直到读到换行符或文件结尾，然后返回空指针。

3.字符串输出

puts()函数、fputs()函数、pirntf()函数
  1. puts()函数使用字符串地址作为参数输出字符串，末尾会加换行符。
  2. fputs()函数是puts()针对文件定制的版本。

4.字符串函数

  1. strlen()函数返回字符串长度。
  2. strcat()用于拼接字符串。
  3. strncat()函数解决strcat()函数无法检查第1个数组是否能容纳第2个字符串。
  4. strcmp()函数用于字符串值的比较。（只比较字符串，如果是数组形式，忽略空字符）
  5. strncmp()函数比较两个字符串时，可以比较到字符不同的地方，也可以只比较第3个参数指定的字符数。

  6. strcpy()和strncopy()函数拷贝整个字符串。把字符串拷贝至临时数组，再从临时数组拷贝至目标数组中。

  7. sprintf()函数把数据写入字符串。可以把多个元素组合成一个字符串。
