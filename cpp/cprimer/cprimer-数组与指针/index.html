CPrimer - 数组与指针

CPrimer - 数组与指针

解引用未初始化的指针

千万不要解引用未初始化的指针
int *pt;	// 未初始化的指针
*pt = 5;	// 严重的错误

第二行的意思是把5储存在pt指向的位置，但是pt未被初始化，其值是一个随机值，所以不知道5储存到何处，可能会擦写数据或代码。

const关键字修饰数组或指针

  1. const修饰的数组无法改变值（编译器报错）;

  2. 指向const的指针不能用于改变值，但是可以让指针指向数组的别处;

    double rates[5] = {88.99, 100.12, 59.45, 183.11, 340.5};
const double * pd = rates;	// pd指向数组的首元素
*pd = 29.89;	// 不允许，指针不能用于改变值
pd[2] = 222.22;	// 不允许，同上
rates[0] = 99.99;	// 允许，因为rates未被const限定
pd++;	// 让pd指向rates[1]，允许

  3. 把const数据或非const数据的地址初始化为指向const的指针或者赋值给指向const的指针是合法的；

    double rates[5] = {88.99, 100.12, 59.45, 183.11, 340.5};
const double locked[4] = {0.08, 0.075, 0.0725, 0.07};
const double * pc = rates;	// 把非const数据初始化为指向const的指针，有效
pc = locked;	// 把const数据赋值给指向const的指针，有效
pc = &rates[3];	// 把非const数据赋值给指向const的指针，有效

  4. 只能把非const数据的地址赋给普通指针；

    double rates[5] = {88.99, 100.12, 59.45, 183.11, 340.5};
const double locked[4] = {0.08, 0.075, 0.0725, 0.07};
double * pnc = rates;	// 把非const数据初始化为普通指针，无效
pnc = locked;	// 把const数据赋值给指向普通指针，无效
pnc = &rates[3];	// 把非const数据赋值给普通指针，有效

  5. const初始化一个不能指向别处的指针（可以修改指针指向的值，但是只能指向初始化时设置的地址）；

    double rates[5] = {88.99, 100.12, 59.45, 183.11, 340.5};
double * const pc = rates; // pc指向数组的开始
pc = &rates[2];	// 不允许，因为该指针不能指向别处
*pc = 92.99; // 没问题--更改rates[0]的值

  6. 创建指针时可以使用两次const，该指针既不能更改它所指向的地址，也不能修改指向地址上的值；

    double rates[5] = {88.99, 100.12, 59.45, 183.11, 340.5};
const double * const pc = rates;
pc = &rates[2]; // 不允许
*pc = 92.99; // 不允许

指针和多维数组

地址的地址或指针的指针就是双重间接的例子

#include <stdio.h>

int main(void) {
  	int zippo[4][2] = {{2, 4}, {6, 8}, {1, 3}, {5, 7}};
    printf("zippo = %p, zippo + 1 = %p\n", zippo, zippo + 1);
    printf("zippo[0] = %p, zippo[0] + 1 = %p\n", zippo[0], zippo[0] + 1);
    printf("*zippo = %p, *zippo + 1 = %p\n", *zippo, *zippo + 1);
    printf("zippo[0][0] = %d\n", zippo[0][0]);
    printf("*zippo[0] = %d\n", *zippo[0]);
    printf("**zippo = %d\n", **zippo);
    printf("zippo[2][1] = %d\n", zippo[2][1]);
    printf("*(*(zippo+2)+1) = %d\n", *(*(zippo + 2) + 1));
}

输出：

zippo = 0x7ffeefbff4f0, zippo + 1 = 0x7ffeefbff4f8	// zippo指向8个字节对象，+1等于地址+8
zippo[0] = 0x7ffeefbff4f0, zippo[0] + 1 = 0x7ffeefbff4f4 // zippo[0]指向4个字节对象，+1等于+4
*zippo = 0x7ffeefbff4f0, *zippo + 1 = 0x7ffeefbff4f4 // *zippo 等同于 zippo[0]
zippo[0][0] = 2
*zippo[0] = 2
**zippo = 2
zippo[2][1] = 3
*(*(zippo+2)+1) = 3

指向多维数组的指针

int (*pz)[2]; // pz指向一个内含2个int类型值的数组，[]优先级高于*，所以需要()
int * pax[2]; // pax是一个内含2个指针元素的数组，每个元素都指向int的指针

指针的兼容性

int n = 5;
double x;
int * p1 = &n;
double * pd = &x;
x = n;	// 隐式类型转换，数值类型之间
pd = p1;	// 编译时错误
// 更复杂的类型
int * pt;
int(*pa)[3];
int ar1[2][3];
int ar2[3][2];
int **p2; //一个指向指针的指针
pt = &ar1[0][0]; // 都是指向int的指针
pt = ar1[0]; // 都是指向int的指针
pt = ar1; // 无效
pa = ar1; // 都是指向内含3个int类型元素数组的指针
pa = ar2; // 无效
p2 = &pt; // both pointer-to-int *
*p2 = ar2[0]; // 都是指向int的指针
p2 = ar2; // 无效

多重解引用让人费解

int x = 20;
const int y = 23;
int * p1 = &x;
const int * p2 = &y;
const int **pp2;
const int n = 13;
p1 = p2; // 不安全，把const指针赋给非const指针，这样可以使用新的指针改变const指针指向的数据，编译器警告
p2 = p1; // 有效，把非const指针赋给const指针
pp2 = &p1; // 不安全，嵌套指针类型赋值，导致const限定符失效，const表示不能通过*pp2修改它所指向的内容
*pp2 = &n; // 有效，两者都声明为const，但是将导致p1指向n（*pp2已被修改）
*p1 = 10; // 有效，但是这将改变n的值（n的值不应该被修改）

C const 和 C++ const

区别是：

  1. C++允许在声明数组大小时使用const整数，而C不允许；

  2. C++的指针赋值检查更严格；

    const int y;
const int * p2 = &y;
int * p1;
p1 = p2; // C++中不允许，但是C中只给警告
// c++不允许把const指针赋给非const指针，但是C允许

函数和多维数组

  1. 定义函数接收二维数组

    void function(int (*pt)[4]); // 接收4列n行的二维数组
// 当且仅当pt是一个函数的形式参数时，可以这样声明：
void function(int pt[][4]); // 第一个方括号是空的，表示pt是一个指针。

  2. 声明一个指向N维数组的指针

    int sum4d(int ar[][12][20][30], int rows); // 只能省略最左边方括号中的值
int sum4d(int (*ar)[12][20][30], int rows);

变长数组

int sum2d(int rows, int cols, int ar[rows][cols]); // 允许
int sum2d(int ar[rows][cols], int rows, int cols); // 不允许，变量rows，cols未定义

复合字面量

复合字面量：基本类型的常量，数组的匿名数组

(int [2]){10, 20}; // 复合字面量
(int[]){50, 20, 90}; // 可以省略数组大小，内含3个元素的复合字面量
// 因为复合字面量是匿名的，所以只能创建的同时使用它
int *pt1;
pt1 = (int[2]){10, 20}; // 复合字面量的类型名也代表首元素的地址，所以可以把它赋给指向int的指针

可以把这种用法应用于多维数组

int (*pt2)[4]; // 声明一个指向二维数组的指针，该数组内含2个数组元素
// 每个元素是内含4个int类型值的数组
pt2 = (int [2][4]) {{1, 2, 3, -9}, {4, 5, 6, -8}};
// 复合字面量的类型是2 x 4的int数组
